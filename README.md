# 💻 Cpp-00-09

This repository contains all C++ modules from the 42 curriculum, from **CPP00 to CPP09**.

Each module is organized in its own folder and contains multiple exercises.

---

## 📁 Project Structure

```
Cpp-00-09/
├── CPP00/
│   ├── ex00/
│   ├── ex01/
│   └── ...
├── CPP01/
├── CPP02/
...
├── CPP09/
```

---

## 📖 Modules Overview

### CPP00 – Basics
Introduction to C++ syntax, namespaces, classes, member functions, and basic I/O.

### CPP01 – Memory & References
Dynamic memory allocation, references, pointers, and file streams.

### CPP02 – Ad-hoc Polymorphism
Operator overloading and canonical class form.

### CPP03 – Inheritance
Introduction to inheritance and class relationships.

### CPP04 – Polymorphism
Subtype polymorphism, virtual functions, and abstract classes.

### CPP05 – Exceptions
Exception handling using try/catch and custom exceptions.

### CPP06 – Casting
Different types of casting in C++ (`static_cast`, `dynamic_cast`, etc.).

### CPP07 – Templates
Function and class templates for generic programming.

### CPP08 – Containers
Working with STL containers and iterators.

### CPP09 – Algorithms
Using STL algorithms and solving more complex problems.

---

## 📦 Usage

Each exercise contains its own **Makefile**.

To run an exercise:

1. Navigate to the exercise folder  
2. Compile using `make`  
3. Run the generated executable  

Example:

```bash
cd CPP00/ex00
make
./<executable>
```
